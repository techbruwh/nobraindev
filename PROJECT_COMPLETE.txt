Step 1: Install Dependencies (First Time Only)
-----------------------------------------------
$ cd /mnt/c/Users/llois/app/snippet-vault
$ chmod +x *.sh
$ ./setup.sh

This will install:
  - Rust programming language
  - Tauri CLI
  - System libraries (webkit2gtk, etc.)

Time: ~10-15 minutes (depends on internet speed)


Step 2: Verify Installation
----------------------------
$ ./verify.sh

This checks if everything is set up correctly.


Step 3: Run the App!
---------------------
$ ./dev.sh

The app will compile and open in a new window!
(First build takes ~5-10 minutes, then it's fast)


Step 4: Create Your First Snippet
----------------------------------
1. Click "New Snippet" button
2. Add title: "Hello World"
3. Select language
4. Paste your code
5. Click "Save"
6. Try searching for it!


Step 5 (Optional): Enable AI Search
------------------------------------
1. Click on "No model" status in the search bar
2. Download TinyLlama (~600MB) or Phi-2 (~1.5GB)
3. Load the model
4. Click the "AI" button in search
5. Search with natural language!
